# Simple Calculator

A calculator program that takes two numbers as input from the user and performs basic arithmeitc.

Addition (+): Takes two numbers from the user and returns their sum to two decimal places.
Subtraction (-): Takes two numbers from the user and returns their difference to two decimal places.
Multiplication (*): Takes two numbers from the user and returns their product to two decimal places.
Division (/): Takes two numbers from the user and returns their quotient to two decimal places.

## Get Started:

Clone this repo and run `gcc -Wall -Werror -Wextra -pedantic -std=gnu89 calculator.c -o calculator` in the same folder to compile and create the executable `calculator` file. Run the file with `./calculator` and follow the prompts to proceed with your calculations.
